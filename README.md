# 🏏 LivewirePRO – Cricket Auction Software

**LivewirePRO** is a powerful and easy-to-use **Cricket Auction Management Application** designed to simplify player auctions for tournaments, leagues, clubs, and cricket communities.

Whether you're organizing a **Box Cricket League**, **Corporate Tournament**, or **Local Premier League**, LivewirePRO helps you run a smooth and exciting player auction.

👉 Learn more: https://livewirepro.in

---

## 🚀 Features

LivewirePRO is built to make cricket auctions **simple, fast, and engaging**.

### Core Auction Features

- 🧑‍🤝‍🧑 Team Management
- 🏏 Player Registration
- 📥 Import Players via CSV
- 💰 Live Player Bidding
- 📊 Real-time Auction Dashboard
- 📈 Auction Analysis & Insights
- 📄 Export Auction Results (PDF)

### Advanced Features

- ⚡ Auto Increment Bidding
- 🧩 Custom Player Fields
- 📤 Export Players List
- 📧 Email Notifications
- ⚙️ Powerful Auction Settings
